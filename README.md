# Track It

An easy-to-use habits manager. Track your habits to spend less time thinking about what to do and don`t forget anything.


## About

This is a web application with which lots of people can manage their own habits. Below are the implemented features:

- Sign Up
- Login
- List all habits
- Add habit
- Remove habit
- Streak
By using this app any user can see your streak and always keep track of your habit.

## How to run

1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Run the front-end with
```bash
npm start
```
4. You can optionally build the project running
```bash
npm run build
```
5. Finally access http://localhost:3000 on your favorite browser
